Projects

  • Picasso Art Bot - A contour mapping & tracing robot using Raspberry Pi.

    Timeline : Jul 2022
    Developed a Python-based interface that accepts a photo and feeds it to a multi-threaded turtlesim bot in a ROS simulation that maps the contour of the given input image and then commands a hardware-implemented 4-wheeled droid built using a raspberry pi microprocessor that draws the contour as a sketch on a piece of canvas.

  • Pneumonia and COVID-19 detection using Convolutional Neural Network (CNN)

    Timeline : Jun 2022
    Developed a deep learning model based on CNN (Convolutional Neural Networks) capable of identifying pneumonia and COVID-19 based on X-ray images.

  • Chess Game using Socket Programming.

    Timeline : May 2022
    A networking project that uses socket.io library from python that connects sockets of two hosts to play multiple chess games using a GUI developed using pygame framework.

  • Sudoku Validator using Multi-Threading.

    Timeline : Dec 2021
    Through the use of Multithreading, this project validates whether the given matrix is a sudoku puzzle solution or not. A total of four models are implemented and compared: the 1-thread model, the 3-thread model, the 11-thread model, and the 27-thread model.

  • Car Price Predictor

    Timeline : Nov 2021
